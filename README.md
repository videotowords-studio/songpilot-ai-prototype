# SongPilot AI Music Prototype

AI 音乐生成网站的可视化交互原型。

## 查看方式

上传到 GitHub 仓库后，在仓库的 Settings -> Pages 中启用 GitHub Pages，入口文件为根目录的 `index.html`。

## 原型内容

- 多页面静态站结构，每个页面都有独立 HTML URL
- 共享 `assets/styles.css` 和 `assets/app.js`
- MakeSong 风格的暗色网格界面、左侧工具导航、顶部胶囊导航
- Home、Create Song、Music Video、Singing Video、Create Photo、AI Dancing Video、Lyrics Video、Seedance 2.0
- Stem Splitter、Vocal Remover、Music Prompt、Projects、Credits、License Center、Account、Pricing、Contact
- 前端多语言切换：中文 / English
- 前端登录态原型：登录弹窗、演示登录、退出登录、本地 session
- credits 报价、生成进度、结果区、失败返还和授权提示
